如何搭建一个自己的博客

Posted by 会飞的星阳 on 2019-03-15

作为一个对技术向往，且有一颗分享之心的程序员都会把自己一些学习心得和问题贴在博客上，例如CSDN、博客园这些平台发表文章还有审核阶段，不如动动小手搭建自己的平台。这里我会一一讲解搭建过程。

不多bb，开讲啦

以下几点搭建个人博客必用

  1. 你需要有一个属于自己的Github账号
  2. 掌握前端知识（稍微掌握也可以，Hexo或者Jekyll有傻瓜式主题）
  3. 需要使用Git bash
  4. 真的好简单，只有三点必用

Github的关键就是可以用Git来控制，这是一个版本工具（我的理解是一个每次上传都备份一个版本，并且可以协调多人同时修改的公用网盘）。当在确定自己的终端可以使用 git –version 命令，并且安装了SSH-Key 之后，你就可以使用如下的基本命令来控制你的库了：
官方文档如何绑定SSH

Git基础命令

git clone 你的SSH地址 （下载你的reposity，这个地址可以用网页浏览的Clone-or-Download-use SSH按钮看到，比如说git@github.com:uername/reposity.git）
git pull （联系服务器，更新本地文件夹）
git add --all （写入本地的所有新文件）
git commit -am "随便说点啥" （在本地进行一次有记录地更新，包括本地所有文件的修改）
git push origin master （把本地的更新推送到服务器

现在离一个好网站只差一步了，就是把某个好网站的所有文件拷贝到你的库里面。
GitHub官方建议你使用博客生成工具 Jekyll。GitHub在库的设置里面还提供Jekyll主题选择。而我的博客使用了Hexo生成工具， 使用它的好处是：1、官方有文档.2.简介明了。Hexo官方文档，超级简单
强烈推荐使用网站工具，这样可以直接用markdown格式编辑文章，而且使用简单的命令实现预览，生成和发布，从而节省大量的时间。

最后

开始创作吧勇士！
